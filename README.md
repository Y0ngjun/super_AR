# super_AR
Draw AR using OpenCV

# Draw CV on chessboard
"CV"단어를 체스판 위에 그리는 프로그램입니다.
입체 도형을 뒷면 > 옆면 > 앞면 순으로 그려 입체감이 더욱 있게 그립니다.
